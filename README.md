# 🤖 Bot de Agendamento Manicure via WhatsApp

## Como rodar

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Rode o bot:
   ```bash
   node index.js
   ```

3. Escaneie o QR Code com o WhatsApp do celular (o número que vai atender os agendamentos).

## Funcionalidades

- Menu interativo
- Seleção de serviços
- Horários disponíveis com bloqueio automático (rolamento)
- Agendamento simples
- Banco SQLite local
